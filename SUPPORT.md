# Support

## Getting Help

Use the repository support paths based on the type of request:

- bug reports: open a GitHub issue with the bug template
- feature requests: open a GitHub issue with the feature template
- security issues: follow [SECURITY.md](SECURITY.md)
- setup and usage questions: use the docs first, then open an issue with enough context

## Before Opening An Issue

Please include:

- framework version
- deployment target
- broker and auth mode
- steps to reproduce
- relevant logs or screenshots

## Scope

This repository focuses on the framework itself:

- workflow engine
- dashboard and Telegram operator surfaces
- plugin system
- deployment guidance

It does not provide financial advice or guarantee trading outcomes.
